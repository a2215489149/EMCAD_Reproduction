# EMCAD Reproduction Package

This repository keeps the code changes, final report, and the curated experiment outputs used in the report.

## Included
- `report/`: final LaTeX report, PDF, and figure
- `experiment_results/ClinicDB/`: baseline and ACMS logs used in the report
- `experiment_results/Synapse/`: official Synapse manual test JSON used in the report
- `experiment_results/experiment_summary.csv`: summary metrics table

## Excluded
- Datasets
- Pretrained weights
- Training checkpoints
- Prediction dumps
- Temporary build outputs
